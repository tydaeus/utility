Linking

Linking within the current "application" in AngularJS should be done using $location.path(), using only the angular-specific portion of the path. If you want to do things the hard way, you could probably also use location.hash.

Linking to another "application" outside the current AngularJS app can be done using a standard anchor (<a>) tag with href set to the URI of the destination "application", or by using methods or properties on the location object, such as location.pathname.
